# ボイスメッセージ送受信アプリ

このプロジェクトは、Flutter製モバイルアプリとNode.js（Express）バックエンドで構成される、音声メッセージの送受信アプリです。

## ディレクトリ構成

- backend/ : Node.js（Express）APIサーバー
- mobile/  : Flutterモバイルアプリ
- .github/ : プロジェクト管理用

## セットアップ手順

### 1. バックエンド
```
cd backend
npm install
npm start
```

### 2. モバイルアプリ
```
cd mobile
flutter pub get
flutter run
```
